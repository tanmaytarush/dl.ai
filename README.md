## Overview

This repository consolidates coursework from multiple Coursera programs focused on machine learning, deep learning, and the mathematical foundations of data science. It is intended for **personal study**, reference, and portfolio use. All materials are used in accordance with course terms; no redistribution of proprietary content is implied.

---

## Repository structure

| Directory | Description |
|-----------|-------------|
| **Coursera-Machine-Learning-Specialization-main** | Supervised learning (regression, classification), advanced learning algorithms, unsupervised learning, recommenders, reinforcement learning. |
| **coursera-deep-learning-specialization-master** | Neural networks and deep learning, improving DNNs (hyperparameters, regularization), structuring ML projects, CNNs, sequence models (RNNs, attention, NLP). |
| **Mathematics-for-Machine-Learning-and-Data-Science-Specialization-Coursera-main** | Linear algebra, calculus, and probability/statistics for ML and data science. |

## Contents by specialization

### 1. Machine Learning Specialization (Coursera)

- **Course 1 — Supervised Machine Learning: Regression and Classification**  
  Regression, cost functions, gradient descent, classification with logistic regression, overfitting and regularization.
- **Course 2 — Advanced Learning Algorithms**  
  Neural networks, vectorization, activation functions, multi-class classification, advice for applying ML.
- **Course 3 — Unsupervised Learning, Recommenders, Reinforcement Learning**  
  Clustering (e.g. K-means), anomaly detection, recommender systems, reinforcement learning (optional).

### 2. Deep Learning Specialization (Coursera)

- **C1 — Neural Networks and Deep Learning**  
  Logistic regression as a neural network, shallow and deep NNs, forward/backward propagation.
- **C2 — Improving Deep Neural Networks**  
  Hyperparameter tuning, regularization (L2, dropout), optimization (momentum, Adam), batch norm.
- **C3 — Structuring Machine Learning Projects**  
  ML strategy, error analysis, train/dev/test sets, transfer learning, multi-task learning.
- **C4 — Convolutional Neural Networks**  
  CNNs, ResNets, object detection, face recognition, neural style transfer.
- **C5 — Sequence Models**  
  RNNs, LSTM, word embeddings, attention, machine translation, trigger word detection.

### 3. Mathematics for Machine Learning and Data Science (Coursera)

- **C1 — Linear Algebra**  
  Vectors, matrices, linear transformations, eigenvalues, PCA.
- **C2 — Calculus**  
  Derivatives, gradients, chain rule, optimization.
- **C3 — Probability & Statistics**  
  Probability distributions, estimation, hypothesis testing, regression from a probabilistic view.

## Prerequisites

- **Python 3.8+** (recommended: 3.10).
- **Jupyter** (for notebooks).
- **Libraries**: NumPy, Matplotlib, Pandas, Scikit-learn; for deep learning: TensorFlow or PyTorch (as required by each course).
- Optional: virtual environment (e.g. `venv`, `conda`) to isolate dependencies.

---

## Usage

1. **Clone the repository**
   ```bash
   git clone https://github.com/tanmaytarush/dl.ai.git
   cd dl.ai
   ```
2. **Navigate** to the relevant course/week folder (e.g. `coursera-deep-learning-specialization-master/C1 - Neural Networks and Deep Learning/`).
3. **Run notebooks** with Jupyter:
   ```bash
   jupyter notebook
   ```
   or open `.ipynb` files in VS Code / Cursor.
4. **Datasets and assets**  
   Some assignments expect data files (e.g. `.h5`, images). Obtain them from the official course pages or course-provided links; they are not included here due to size and licensing.
